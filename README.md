# 描述
-----
项目主要目的是在用户登录，注册场景下介绍如何使用sinon来实施

### 项目环境说明
Node.js  5.8.0
更多依赖信息查看项目的package.json文件

### 组件
语法检查： eslint
文档：apidoc
日志：log4js, 日志会统一打到/var/log目录下
调试：nodemon
Http框架：koa
javascript版本：es6
测试框架：mocha
测试组件：sinon
测试断言: assert
database: mysql

### 如果启动
开发调试：npm start
测试：npm test

生产部署：make online
测试部署：make test
